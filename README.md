## Overview
This project demonstrates a simple Go setup, including installation, module initialization, and a Hello World program.

## Requirements
- Go 1.20+
- VS Code or similar editor

## Setup
1. Install Go from https://go.dev/dl/
2. Verify installation:
   go version
3. Initialize module:
   go mod init capstone
4. Run the program:
   go run main.go
